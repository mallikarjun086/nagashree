# 💰 First Salary Guide

> The financial blindspot solver for first-generation earners and fresh graduates. Decode your paycheck, build your plan, and start confident — with zero jargon and zero judgment.

![First Salary Guide Cover](https://via.placeholder.com/1200x600/0d7377/ffffff?text=First+Salary+Guide)

## 🏆 The Problem
People receiving their first formal salary have no mental model of how to allocate it. They don't understand in-hand vs. CTC, PF deductions, or basic saving ratios. Furthermore, asking for help feels embarrassing, creating a socially invisible wall of financial ignorance.

## 💡 Our Solution
**First Salary Guide** is an instant, frictionless, single-session web app that:
1. **Translates CTC to In-Hand:** Automatically applies real Indian Income Tax slabs (FY 2024-25 Old vs. New Regime) to calculate exact take-home pay.
2. **Explains Deductions:** Breaks down PF, PT, and TDS in plain English.
3. **Builds a Budget:** Interactive 50/30/20 sliders let users plan their lifestyle.
4. **Tracks Goals:** A built-in goal tracker shows how fast they can reach milestones based on their savings rate.
5. **AI Money Mentor:** An integrated, context-aware AI chatbot (powered by Gemini) acts as a private, judgment-free zone to ask "dumb" money questions.

## ✨ Key Features that wow
- **Privacy-First (No Login Required):** Operates entirely in a single session using Local Storage. No data hoarding, maximum trust.
- **Progressive Web App (PWA):** Installable on desktop and mobile for offline access.
- **Professional PDF Export:** Generates a structured, text-selectable financial report using `jsPDF`.
- **Dynamic Social Proof:** A live Node.js backend tracks the aggregate number of plans generated by the community.

## 🛠️ Tech Stack
- **Frontend:** HTML5, Vanilla CSS3 (Glassmorphism design), ES6 JavaScript
- **Backend:** Node.js, Express.js
- **AI Integration:** Google Gemini 2.0 Flash-Lite via Server Proxy
- **Libraries:** Chart.js (Data Vis), jsPDF (Exports), Driver.js (Onboarding)

## 🚀 How to Run Locally

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set your Gemini API Key in a `.env` file (or directly in `server.js` for local testing):
   ```
   GEMINI_API_KEY=your_key_here
   ```
4. Start the server:
   ```bash
   node server.js
   ```
5. Open `http://localhost:3456` in your browser.
